# My flake templates
A collection of templates edited/inspired from [nix-community/templates](https://github.com/nix-community/templates/) for my use cases.
## Usage
Replace `<template>` with the name of the template of the language template you want to use
```bash
mkdir project && cd project
nix flake init -t github:MijaToka/nixtemplates#<template>
```
